New warband
Opens the Song of Blades and Heroes Online Warband Creator in a new tab or window where you can design a new warband from scratch.
Load from file
Load a warband from a file that you have previously stored on your local drive.
Save to file
Save your warband to a file on your local drive. May not work in all browsers.
Undo
Undo last change.
Show warband code
Displays the warband code so you can copy it to a text-file and save it.
Print
Shows this page in a printable version, without buttons etc. Pops up a new window.
Settings
Set Settings like rules scope or data language or display mode.

Required permissions for cloud storage services

Here's why OWC requests certain permissions for using cloud storage services.

Microsoft OneDrive

Permission Explanation
Read your profile Essential permission to access your OneDrive. OWC does not read your user name, e-mail or any other profile information.
Open OneDrive files For loading warbands stored in files on your OneDrive. OWC does access only files that you select.
Have full access to all files you have access to For reading and writing warband files on your OneDrive. OWC does access only files that you select. If a <warbandname>.owc.txt-file exists in the folder you select, it will be replaced by the current warband with the same name. OWC does not delete any files.
